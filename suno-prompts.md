## Batch Workflow

1. Open Suno
2. Paste style tag + lyrics for module 1
3. Generate (takes ~30 seconds)
4. Download the MP3
5. Name it `01-cloud-models.mp3`
6. Repeat for all 12
7. Drop all MP3s into `public/songs/` in the repo
8. Update SongPlayer to use `<audio>` tags instead of Web Audio

File naming convention:
```
public/songs/
  01-cloud-models.mp3
  02-shared-responsibility.mp3
  03-regions.mp3
  04-resource-hierarchy.mp3
  05-compute.mp3

```
